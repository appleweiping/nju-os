### M5 frecov verification ###
gcc 13.3. Scans the whole FAT32 image on 32-byte boundaries for directory
records, reconstructs long file names from LFN runs, recovers BMP data from
contiguous clusters, verifies the BM header, and prints '<sha1>  <name>'.
SHA-1 (RFC 3174) is implemented inline.

TEST A: 5 short-named BMPs (16x16..48x48), FAT32 image built with mkfs.fat + mtools.
  live image   -> 5/5 sha1 match the originals (byte-exact)
  after mdel    -> 5/5 sha1 STILL match (data recovered post-deletion)
