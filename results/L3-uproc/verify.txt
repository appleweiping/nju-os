### L3 uproc verification (AbstractMachine native backend, WSL2 Ubuntu 24.04, gcc 13.3) ###
Build:  cd oslabs/user && make   &&   cd oslabs/kernel && make ARCH=native
Run:    smp=<N> ./build/kernel-native

Each user process has its own AM AddrSpace (protect/map/ucontext) over L1 pmm
pages; a syscall traps via the native 0x100000 trap page (EVENT_SYSCALL); the L2
kmt scheduler time-slices user processes.  init (pid 1) drives the suite; the L3
section exercises getpid/uptime/fork/exit/wait/exec/mmap/sleep (the L4 section is
covered in results/L4-vfs).
